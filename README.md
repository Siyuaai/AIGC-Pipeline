本项目是一套基于 Streamlit 与 ComfyUI API 构建的自动化 AI 内容生产工作站。

它并非市面上常见的简单生图工具，而是专门针对 “BI（商业智能）+ AIGC + 游戏/文创” 深度融合场景设计的工程化流水线。我们的核心逻辑是：将碎片化的创意设计，转化为标准化、可量化且能够规模化生产的工业流程。

核心功能
大规模自动化生产线
系统内置 CSV 工单调度逻辑。你只需要在任务清单中定义好 Prompt 与参数，即可实现上百个生图任务的“一键启动、无人值守”式作业，极大提升了内容产出的规模化产能。

商业级 4K 画质交付
采用 Dual-Pass（双重采样）技术架构：先完成基础构图，再通过物理放大与像素级细节精修，配合 4x-UltraSharp 算法，让生成的每一张图片都达到高清商业应用标准。

高精度视觉约束控制
深度集成了 Canny 边缘检测和 OpenPose 姿态识别等多种视觉控制模型，并内置了自动图像预处理器。即便不需要手动处理原图，也能精准把控画面的空间结构与人物动态。

模块化风格管理
支持 LoRA 外挂系统的灵活接入与权重微调。你可以针对不同的项目需求，进行精准的画风定向与强度调节，确保整批产出的视觉风格高度统一。

数据科学与 BI 集成
系统会自动记录每一项任务的 Prompt、种子点、渲染耗时等关键元数据。这些数据不仅是操作记录，更是后续进行生成质量分析、算力成本监控以及 BI 数据闭环的重要原始资产。

技术栈
前端界面：Streamlit (基于 Python 的交互式 UI)

后端引擎：ComfyUI (API 模式无头运行)

架构逻辑：有向无环图 (DAG) 执行逻辑

核心模型：SD 1.5 / Realistic Vision / Anything V5

快速启动
确保本地的 ComfyUI 服务已正常启动。

在 app.py 中配置你本机的模型存放路径。

在 jobs.csv 中按照模板填入你想要处理的任务清单。

在终端执行 streamlit run app.py 即可进入工作站控制面板。
